Key Skills:

  • Python Programming: Developed solutions for diverse tasks, including scientific data visualization, web scraping, and task management APIs.
  • Data Visualization: Utilized libraries like Matplotlib and Seaborn to process and visualize complex datasets, effectively communicating insights from weather, health, and biological data.
  • Web Development: Built small applications using Flask and Django, demonstrating proficiency in creating web-based tools like code repositories and task management APIs.

I enjoy tackling complex challenges and breaking them down into manageable tasks, whether it's building a simple Python calculator or creating a machine learning model.
